# Sudoku en Java

Aplicación desarrollada en Java utilizando programación orientada a objetos que permite generar, resolver y gestionar tableros de Sudoku con distintos niveles de dificultad.

## 🧩 Descripción

El sistema permite tanto jugar Sudoku como generar nuevos tableros automáticamente o ingresar uno manualmente para su resolución. Incluye lógica para validación de movimientos y resolución automática del juego.

## 🚀 Funcionalidades

- Generación automática de Sudokus
- Niveles de dificultad: fácil, normal y difícil
- Ingreso manual de tableros personalizados
- Validación de jugadas según reglas del Sudoku
- Resolución automática del tablero
- Interfaz basada en consola (o gráfica, si aplica)

## 🛠 Tecnologías utilizadas

- Java
- Programación Orientada a Objetos

## 🧠 Conceptos aplicados

- Estructuras de datos (matrices)
- Backtracking (para resolución del Sudoku)
- Encapsulamiento y modularización
- Lógica algorítmica

## 📌 Estado del proyecto

Proyecto funcional desarrollado con fines de práctica y aprendizaje.
